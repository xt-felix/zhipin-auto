# GoodHR AI高级版功能说明

## 功能概述
GoodHR AI高级版在原有免费版的基础上，新增了AI智能筛选功能，可以根据岗位要求智能判断候选人是否合适。

## 主要特性

### 1. 双模式支持
- **免费版**：使用关键词匹配的传统筛选方式
- **AI高级版**：使用AI智能筛选，根据岗位说明进行判断

### 2. AI配置
- 使用轨迹流动AI服务
- 可配置轨迹流动Token和模型参数
- 实时连接状态检测

### 3. 岗位管理
- 统一的岗位列表（免费版和AI版共用）
- 每个岗位可添加详细说明（AI版显示）
- 支持多岗位管理

## 使用步骤

### 1. 配置AI设置
1. 切换到"AI高级版"选项卡
2. 点击右上角"AI配置"按钮
3. 输入API Token（获取方式见下方）
4. 选择AI模型或输入自定义模型名称
5. 点击"保存配置"
   - 可以单独保存Token或模型名称
   - 配置会自动同步到服务器（如果绑定了手机号）

### 2. 添加岗位
1. 在"岗位管理"中输入岗位名称（免费版和AI版共用）
2. 切换到AI高级版，选中岗位后会自动显示"岗位说明"输入框
3. 在"岗位说明"中详细描述岗位要求，包括：
   - 岗位职责
   - 任职要求
   - 技能要求
   - 经验要求
   - 其他要求

### 3. 开始筛选
1. 在招聘平台筛选好基本条件（年龄、学历等）
2. 点击"AI筛选"按钮
3. AI将根据岗位说明智能判断候选人是否合适

## API Token获取方式

### 轨迹流动
1. 访问 https://www.guiji.ai/
2. 注册并登录账号
3. 在控制台获取API Token
4. 复制Token到配置中

## 技术实现

### 前端（Chrome扩展）
- 选项卡切换：免费版/AI高级版
- AI配置管理：Token、模型设置（支持单独保存）
- 岗位管理：AI版本的岗位和说明
- 设置同步：本地存储和服务器同步（包括AI配置）
- 直接调用轨迹流动API
- 智能提示词构建
- 本地AI响应解析
- 统一配置管理：通过config.js统一管理所有API地址和配置

### 筛选逻辑
1. 保持原有的滚动和候选人信息提取逻辑
2. 在筛选阶段判断当前模式：
   - 免费版：使用关键词匹配
   - AI版：直接调用轨迹流动API进行智能判断
3. AI返回"是"或"否"，决定是否打招呼

### 数据结构
- 岗位数据结构统一，包含：name、keywords、excludeKeywords、description
- 免费版：使用keywords和excludeKeywords进行筛选
- AI版：使用description进行智能筛选，岗位说明只在选中岗位时显示

## 注意事项

1. **API费用**：使用AI功能会产生API调用费用，请合理使用
2. **网络要求**：需要稳定的网络连接访问AI服务
3. **Token安全**：请妥善保管API Token，不要泄露给他人
4. **筛选精度**：AI筛选结果仅供参考，建议人工复核
5. **直接调用**：AI请求直接从插件发起，不经过服务器
