# 好工作AI助手

## 项目概述

好工作AI助手是一个浏览器扩展程序，旨在帮助HR专业人员更高效地筛选和评估候选人资料。该扩展支持多个招聘网站，提供免费版和AI版两种模式：

- **免费版**：基于关键词匹配筛选候选人
- **AI版**：利用人工智能技术进行智能筛选和评估

## 核心功能

1. **多平台支持**：适配BOSS直聘、拉勾网、猎聘、智联招聘等多个招聘平台
2. **智能筛选**：
   - 关键词匹配
   - AI智能评估候选人匹配度
3. **自动滚动**：自动浏览候选人列表
4. **候选人高亮**：根据匹配结果对候选人进行颜色标记
5. **简历下载**：支持批量下载候选人简历
6. **岗位优化**：AI辅助优化岗位描述

## 主要组件

- **后台服务**：管理扩展的核心逻辑和数据
- **内容脚本**：负责与招聘网站交互，提取和处理候选人信息
- **弹出界面**：用户配置界面，用于设置筛选条件和查看日志
- **解析器**：针对不同招聘网站的专用解析模块
- **AI集成**：与轨迹流动API集成，实现智能筛选功能

## 使用说明

1. 安装扩展后，在招聘网站的候选人列表页面点击扩展图标
2. 在弹出界面中选择"免费版"或"AI版"模式
3. 配置筛选条件：
   - 输入关键词和排除词
   - 设置滚动延迟和匹配暂停数
   - 启用/禁用提示音
4. 点击"开始"按钮启动自动筛选
5. AI版用户可输入岗位要求，系统将智能评估候选人匹配度

## 开发者说明

本项目采用模块化设计，主要代码结构如下：

- `content_scripts/`：浏览器内容脚本，负责与网页交互
- `content_scripts/sites/`：针对不同招聘网站的具体实现
- `popup/`：扩展弹出界面和相关逻辑
- `background.js`：后台服务逻辑
- `config.js`：配置管理

## 注意事项

1. 使用AI功能会产生API调用费用，请合理使用
2. 需要稳定的网络连接访问AI服务
3. 不同招聘网站的页面结构可能会变化，可能需要定期更新解析器
