# GoodHR - 智能招聘助手浏览器扩展 | 猎头工具 | HR招聘软件

## 🚀 项目简介

GoodHR是一款专为HR和招聘人员设计的智能浏览器扩展，旨在提高招聘效率，简化招聘流程。通过自动化工具和智能功能，帮助招聘人员在各大招聘平台上更高效地完成工作。

作为一款专业的**招聘助手软件**和**猎头工具软件**，GoodHR集成了**自动打招呼**、**简历下载**、**设备指纹识别**等核心功能，是HR招聘人员不可或缺的**招聘神器**。无论您是企业HR、**猎头顾问**还是招聘团队负责人，GoodHR都能显著提升您的工作效率，让招聘变得更加轻松高效。

GoodHR特别针对**猎头工作场景**进行了优化，提供**猎头找人**、**猎头挖人**、**人才搜索**等专业功能，是**猎头软件**、**猎头工具**、**猎头系统**的首选解决方案。

## 📦 安装方法

### 方法一：从Chrome Web Store安装（推荐）
1. 访问Chrome Web Store
2. 搜索"GoodHR"
3. 点击"添加到Chrome"

### 方法二：手动安装
1. 下载本仓库代码
2. 解压到本地文件夹
3. 打开Chrome浏览器，进入`chrome://extensions/`
4. 开启"开发者模式"
5. 点击"加载已解压的扩展程序"
6. 选择解压的文件夹

## 🎮 使用方法

1. **安装扩展**：按照上述方法安装GoodHR扩展
2. **登录账号**：在扩展中登录您的招聘平台账号
3. **选择功能**：根据需要选择相应功能模块
4. **开始使用**：在支持的招聘平台页面上，扩展会自动激活相应功能
