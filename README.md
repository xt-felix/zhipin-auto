# SmartHR Assistant - 智能HR招聘助手

## 📋 项目说明

本项目是基于 [GoodHR AI JS](https://gitee.com/xxooxx8/good-hr-ai-js) v2.8.1 版本进行的二次开发版本。

**原项目地址**: https://gitee.com/xxooxx8/good-hr-ai-js

**开发者**: [xt-felix](https://github.com/xt-felix)

## 🚀 项目简介

SmartHR Assistant 是一款专为HR和招聘人员设计的智能浏览器扩展，旨在提高招聘效率，简化招聘流程。通过自动化工具和智能功能，帮助招聘人员在各大招聘平台上更高效地完成工作。

## ✨ 主要功能

### 1. 智能简历下载
- 一键下载候选人简历，无需手动操作
- 支持批量下载，提高工作效率
- 自动命名和分类简历文件
- 支持多平台简历下载

### 2. 自动打招呼功能
- 自动向候选人发送定制化打招呼消息
- 支持多平台自动沟通
- 智能关键词筛选

### 3. 设备指纹识别
- 基于FingerprintJS v5技术的设备指纹识别
- 防止多账号使用，确保账号安全

### 4. AI智能筛选
- AI高级版支持智能候选人筛选
- 根据岗位要求自动判断候选人匹配度

## 🎯 适配平台

- **Boss直聘** - 支持简历下载、自动沟通、职位管理
- **猎聘** - 支持猎头功能、简历下载、自动打招呼
- **智联招聘** - 支持简历筛选、下载、候选人管理
- **前程无忧(58同城)** - 支持职位发布、简历管理
- **拉勾网** - 支持IT技术岗位招聘功能

## 📦 安装方法

### 手动安装
1. 下载本仓库代码
2. 解压到本地文件夹
3. 打开Chrome浏览器，进入`chrome://extensions/`
4. 开启"开发者模式"
5. 点击"加载已解压的扩展程序"
6. 选择解压的文件夹

## 🎮 使用方法

1. **安装扩展**：按照上述方法安装SmartHR Assistant扩展
2. **登录账号**：在扩展中登录您的招聘平台账号
3. **选择功能**：根据需要选择相应功能模块
4. **开始使用**：在支持的招聘平台页面上，扩展会自动激活相应功能
